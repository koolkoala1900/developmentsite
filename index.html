Development Project: Clothing App

PROJECT PROMPT

In this project, we developed an interactive interface, used interface components, and tied the components to an internal data state. This project allowed me to get comfortable with React by creating a list-based interface for users to select items out of a list, which then aggregates them.

FEATURES WE INCLUDED

At least 2 filtering categories

At least 1 sorting feature

An aggregator section (can be a dashboard, list, cart) on the same page displaying:

Items you added to the aggregator

Aggregation of a property: total time, cost, votes, etc., but NOT the total number of items

At least 12 item cards, each displaying: Item image Categories that your filter uses Field(s) that your sort uses Property you are aggregating A button that adds to/removes from the aggregator
